# Brain Tumor Detection System

An advanced AI-powered Brain Tumor Detection, Segmentation, and Classification system developed using PyTorch and MRI image analysis. This project combines deep learning, medical image processing, and radiomics to assist in accurate tumour identification and analysis from brain MRI scans.

The system integrates multiple components including tumour segmentation, tumour type classification, survival prediction, tumour growth analysis, Grad-CAM visualization, and radiomics-based feature extraction. It supports BraTS 2021 and Kaggle Brain Tumor MRI datasets and includes optimized training pipelines for high-performance GPU execution.

Key features of this project include:

- Brain tumour segmentation using Deep UNet 3+
- CBAM attention modules for improved small tumour detection
- MRI tumour classification using deep convolutional neural networks
- Radiomics feature extraction and analysis
- Grad-CAM explainability visualization
- Survival prediction and tumour growth analysis models
- Optimized PyTorch training pipeline with AMP and OneCycleLR
- Test-Time Augmentation (TTA) for improved inference accuracy
- Synthetic MRI data generation support
- Medical image preprocessing and skull stripping
- Multi-modal MRI support (T1, T1ce, T2, FLAIR)

This repository contains pretrained best-performing model weights, training logs, notebooks, and complete source code implementation for experimentation and research purposes.
